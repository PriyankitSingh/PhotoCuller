# Photo Culler - Project Specification

## Overview
A high-performance, cross-platform desktop application for quickly reviewing and culling paired RAW+JPEG photo files. Built with Rust for fun.

## Problem Statement
Photographers who shoot RAW+JPEG need to quickly review photos and decide which formats to keep or delete. This tool provides a keyboard-driven interface to mark files for deletion and bulk delete them.

## How to build?
Go to repository directory and run the following. 
``cargo run --release``
